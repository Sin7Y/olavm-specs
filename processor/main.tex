\section{Processor}\label{sec: processor}
处理器模块是OlaVM的核心模块之一。它一步一步的执行编译器提供的指令，在执行过程中生成trace。零知识证明者会使用trace生成证明。
处理器模块包含了下边这些模块：
\begin{itemize}
    \item \verb|instructions set architecture(ISA)|
    \item \verb|prophet executor|
    \item \verb|builtin|
    \item \verb|memory|
    \item \verb|trace generator|
\end{itemize}
